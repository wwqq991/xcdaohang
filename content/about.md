# 关于这个导航站

这是一个面向公开分享的个人导航站，目标很简单：

- 别人打开后能直接找到常用网站
- 我自己每天也能把它当作起始页来用
- 后续可以持续加入我整理过的精选网址

## 目前的内容结构

- `常用`：放大多数人都会访问的网站，例如 AI、开发、学习、设计等
- `精选`：放我自己长期整理、愿意推荐给别人的站点
- `关于`：记录这个站点的定位、维护方式和后续更新方向

## 适合怎么维护

这个项目是静态站点，不需要数据库，也不需要后端服务。

日常维护的方式很轻：

1. 在 `config/user/pages/*.yml` 里新增或修改网址
2. 本地预览确认没问题
3. 提交到 Git 仓库
4. 部署到 Cloudflare Pages

## 为什么这样做

我希望这个站不是普通的大杂烩导航，而是：

- 有常见分类，方便别人使用
- 有个人精选，保留自己的整理价值
- 成本低，后续长期维护不会有负担

## 下一步可以继续做什么

- 把你自己常用的网站继续补到 `精选` 页面
- 按你的实际使用习惯细分更多分类
- 换成你的站名、域名和个人介绍
- 部署到 Cloudflare Pages 并绑定自己的域名
